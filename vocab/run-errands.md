# Flash Card: Run Errands

## Front

Мені довелося <u>відлучитися по справах</u>.
Вона пішла <u>зробити справи</u>: купити продукти та забрати одяг з хімчистки.

## Back

<div style="text-align: left;">
  <strong>Translation</strong>: "I've had to be out running errands. She went to run errands: buy groceries and pick up clothes from the dry cleaner."<br /><br />
  <strong>Word</strong>: run errands<br /><br />
  <strong>Explanation</strong>: to go out and do small tasks or chores, such as shopping, picking up items, going to the bank, or other routine activities<br /><br />
  <strong>Examples</strong>:<br />
  I'm going to run errands this afternoon - need to stop by the post office and pharmacy.<br />
  Can you run some errands for me while you're downtown?<br />
  She spent the morning running errands around town.<br />
  We need to run errands before the weekend starts.
</div>
